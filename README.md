## Naming Conventions
<pre>
  - Filenames: underscore_separated_lowercase
  - Code:
    - HTML/CSS ids & classes: hyphen-separeted
    - JS variables & functions: lowerCamelCase
    - JS Classes: UpperCamelCase
    - SQL:
      - tables: underscore_separated_lowercase
      - columns: lowerCamelCase
</pre>
